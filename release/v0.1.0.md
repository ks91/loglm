## loglm v0.1.0

First public release of `loglm`: a logged launcher for Codex, Claude, and Gemini, with prompt-agent install support.

### Highlights

- Launch AI coding agents with terminal logging via `script`
- Per-run log files by default:
  - `logs/loglm-<agent>-log-YYYYMMDD-HHMMSS-pid<PID>.txt`
- Decode command:
  - `loglm-decode <raw-log-file>`
- Prompt-agent commands:
  - `loglm agent install|list|remove|update`
- Prompt-agent install flow:
  - source resolution from `AGENT_INSTALL*.md` (first-match, no merge)
  - local prompt file as `<REPO-NAME-UPPER>.md`
  - managed reference blocks in `AGENTS.md` / `CLAUDE.md` / `GEMINI.md`
  - auto-start current coding agent in a new context
- Gemini default launch now uses `gemini --resume`

### Platform Status

- Tested:
  - macOS
  - Ubuntu
  - Ubuntu on Lima (macOS)
  - Ubuntu on WSL2
- Untested:
  - Raspberry Pi OS
  - Chrome OS (Linux container / Crostini)

### Notes

- Claude Code: after Homebrew/npm install, native install is recommended.  
  Run `claude install` to migrate to native installation.
- Gemini resume behavior can vary by environment; if context is not restored, run `/resume` or `/chat resume` inside Gemini.

### Install / Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/uninstall.sh | bash
```
