## loglm v0.1.4

This release focuses on log post-processing for research and review workflows.

### Highlights

- Expanded `loglm-decode`
  - trims overlapping leading context by default
  - improves Claude Code log cleanup
  - improves Gemini CLI log cleanup
  - adds grouped PII redaction workflow based on `pii-candidates.txt`
  - writes anonymized output as `*.redacted.txt`
- Added `PII_REDACTION_GUIDE.md`
  - bilingual workflow guide (English / Japanese)
  - recommended practice for preparing research-safe log exports
- Added `loglm-timeline`
  - extracts compact session timelines from decoded or redacted logs
  - sorts sessions by log start time
  - helps reconstruct cross-session work history

### Notes

- `loglm-decode --review-pii` now expects a grouped `pii-candidates.txt` file as the first positional argument.
- `loglm-timeline` is a first practical release. It is designed for general timeline extraction, not domain-specific semantic labeling.

### Install / Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/uninstall.sh | bash
```
