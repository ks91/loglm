## loglm v0.1.3

This release focuses on platform coverage and installation reliability.

### Highlights

- Fixed Node.js / npm setup on Chrome OS when `NVM_DIR` is set but the directory does not exist yet
- Added a regression test covering missing-`NVM_DIR` install flow
- Updated platform status:
  - Raspberry Pi OS is now tested
  - Chrome OS (Linux container / Crostini) is now tested

### Notes

- Chrome OS support assumes Linux container (Crostini) usage.

### Install / Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/uninstall.sh | bash
```
